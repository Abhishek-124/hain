You can use this interface for adding or removing Result entries. This interface is always provided as the second argument to Plugin.search().

Example

function search(query, res) {
     // Add an entry to the result set
     res.add({
         id:    'temp',
         title: 'Fetching...',
         desc:  'Please wait a second',
     });

     // Remove entry 'temp' after `1000ms`
     setTimeout(() =>
         res.remove('temp'),
     1000);
}

  • Add a SearchResult to the result-set

  • remove(id: string): void

  • You can remove a SearchResult from the result-set that you previously added
